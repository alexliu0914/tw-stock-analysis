# 🚀 快速部署指令

## GitHub Pages 部署（推薦）

```powershell
# 1. 進入專案目錄
cd "d:\Zyxel\python練習\Irene_tw_stock"

# 2. 初始化 Git
git init

# 3. 添加所有檔案
git add .

# 4. 提交
git commit -m "台股分析工具 v1.1 - 新增圖表功能"

# 5. 連接到 GitHub（替換成你的倉庫網址）
git remote add origin https://github.com/你的用戶名/tw-stock-analysis.git

# 6. 推送到 GitHub
git branch -M main
git push -u origin main
```

## 啟用 GitHub Pages

1. 進入 GitHub 倉庫
2. 點擊 `Settings`
3. 左側選單找到 `Pages`
4. Source 選擇 `GitHub Actions`
5. 等待 1-2 分鐘部署完成

## 訪問網址

```
https://你的用戶名.github.io/倉庫名稱/
```

## 測試功能

1. 輸入股票代號（例如：2330）
2. 點擊「分析」
3. 查看圖表（向下滾動）
4. 測試互動功能

## 手機使用

1. 在手機瀏覽器開啟網址
2. 點擊「加入主畫面」
3. 像 App 一樣使用
