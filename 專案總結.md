將所有檔案複製到手機，用瀏覽器開啟 `index.html`
（注意：需要網路連線才能獲取股票數據）

## ⚠️ 已知限制

1. **CORS 限制**: 本地運行時無法直接訪問 Yahoo Finance API
   - **解決方案**: 部署到網路主機或使用 CORS 代理

2. **API 限制**: Yahoo Finance 可能有查詢頻率限制
   - **影響**: 大量掃描時可能較慢

3. **數據延遲**: 使用免費 API，數據可能有延遲
   - **影響**: 非即時報價
